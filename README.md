# SabanciInsight
## Description
SabancıInsight is a mobile application designed specifically for Sabancı University students to make more informed course selection decisions. Unlike traditional course registration tools, this app provides detailed insights about courses, instructors, workload, grading style, and student experiences. It acts as a hybrid between a course planner and a professor/course review platform.
